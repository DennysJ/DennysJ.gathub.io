Tipos de Redes

Tipos de Redes Definición Función
Redes LAN. ("Local Area Network/Red de Área Local”) Una LAN es una red que conecta computadoras y dispositivos dentro de un área limitada, como una oficina, un edificio o una escuela. Las LAN permiten compartir recursos, como archivos e impresoras, y suelen utilizar cableado Ethernet o tecnologías inalámbricas como Wi-Fi. La red de una oficina o escuela donde las computadoras están conectadas a través de cables o Wi-Fi.
Redes MAN. ("Metropolitan Area Network /Red de Área Metropolitana”) Una MAN es una red que cubre un área geográfica más grande que una LAN pero más pequeña que una WAN, como una ciudad o una gran campus. Se utiliza para conectar varias LAN en ubicaciones diferentes dentro de una misma área metropolitana. Los proveedores de servicios de telecomunicaciones suelen implementar MAN. La red que conecta varios edificios de una universidad dentro de una ciudad.
Redes WAN. ("Wide Area Network/Red de Área Amplia”) Una WAN conecta redes en grandes áreas geográficas, como países o continentes. Las WAN suelen utilizar enlaces de telecomunicaciones como líneas alquiladas o satélites. Internet, que conecta redes de todo el mundo
Redes PAN. ("Personal Area Network/Red de área personal"): Es una red pequeña que conecta dispositivos cercanos, generalmente en un radio de unos pocos metros. Se utiliza para conectar dispositivos como teléfonos móviles, tabletas, auriculares y otros dispositivos personales, usando tecnologías como Bluetooth o USB. Bluetooth entre un teléfono móvil y unos auriculares inalámbricos.
