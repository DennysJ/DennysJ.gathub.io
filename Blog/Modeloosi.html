Modelo OSI

El Modelo OSI (de las siglas en inglés: Open Systems Interconnection, o sea, “Interconexión de Sistemas Abiertos”), es un modelo de referencia para los protocolos de comunicación de las redes informáticas o redes de computadores. Fue creado en la década de 1980 por la Organización Internacional de Normalización (ISO).

Estandariza las funciones de una red de comunicaciones en siete capas. Cada capa tiene responsabilidades específicas y facilita la interoperabilidad entre sistemas diferentes. Aquí tienes un resumen:

Capa de Aplicación: Proporciona servicios de red a las aplicaciones de software (como correo electrónico, navegación web, etc.).

Capa de Presentación: Traducir datos entre la capa de aplicación y el formato de transmisión, asegurando que los datos sean comprensibles.

Capa de Sesión: Gestiona las conexiones y las sesiones de comunicación entre aplicaciones.

Capa de Transporte: Asegura la transferencia fiable y ordenada de datos entre dispositivos finales.

Capa de Red: Gestiona la dirección y el enrutamiento de los paquetes de datos entre nodos en diferentes redes.

Capa de Enlace de Datos: Se encarga de la transferencia de datos entre dos nodos conectados directamente, controlando el acceso al medio físico.

Capa Física: Define los medios físicos de transmisión y la interfaz con el hardware.
